WORK ABOUT RÉSUMÉ

Web Configurator

Installation Package Generator

June 2015

Overview

This was a new product requireing an MVP.
OEMs developing for IoT requested an online method for creating their own cross-platform bundles that would be used for offline installation scenarios.

Problem

We have an extensive suite of development tools for OEMs that are bringing up new systems. The file sizes are quite large so downloadingthe entire suite for all host and target OS combinaitons takes significaant time, space and bandwidth. Define the Problem and substantiate it System Bring Up Tools Advanced Installer offers support for creating an EXE setup file that will download the MSI and CAB files Use the web-based installer to select products from a suite of products, instead of downloading all products in that suite, saving you time and bandwidth. You can also download and save an installer for later installation. Market research of internal business metrics

Solution & Key learnings

An online configurator thats easy to use and generates installer packages for various OS host and target permutations.

Challenges

  • Short design and development cycle for our team
  • Completely new concept
  • systems should facilitte both public and NDA tools
  • Many tool and platform combinations are not compatible. The system must handle gracefully
  • Primary user base is in Asia

Role and Team

User flows, Information Architecture, Sketching , Wireframing, UI, Prototyping, Set up for user study
There is clarity in roles and responsibilities. Members know what is expected and what falls within their realm of influence.
