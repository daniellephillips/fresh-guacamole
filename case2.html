Meadow Mountain

Device Manager

June 2017

Product Overview

Meadow Mountain was the code name for a free cloud-based IDE built to support the rapid prototyping phase for IoT developers. Features included fully managed toolchains, an editor, debugger, and a device manager for managing remote IoT devices.

Role and Challenges

My role was UX design lead and I worked with a UX researcher. engaging with various stakeholders across business groups. I managed the UX backlog and worked with the engineering team during sprints. The initial release called for MVP features that would accommodate IoT prototyping use cases.

The device manager was a value-added feature that would eventually break off into its service and product. Realizing and accounting for the security requirements presented the biggest challenge. From a usability perspective, a wrong configuration setting due to a confusing UI could make customers vulnerable, so it was essential to get it right. I worked with engineering architects to understand device registration, security controls and data encryption, then translated those findings to UX requirements for the device registration flow.
